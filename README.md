# Backend for Electronic Device Repair and Refurbishment

## Description

This backend is developed for a company that repairs and refurbishes electronic devices.  
The system allows the addition of endpoints as needed, in addition to those specified in the project requirements.  

The application is used by two types of users:  
- **Managers (or Admins):** Manage parts inventory and user accounts.  
- **Technicians:** Perform repairs and update the status of devices.
